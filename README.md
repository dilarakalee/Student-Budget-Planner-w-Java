üöÄ Features

- Custom UI Components: Uses custom RoundedButton and RoundedTextField classes for a modern look.
- SQLite Database: All data is persistently stored in a local proje.db file. Data is not lost when the application is closed.
- Dynamic Themes: Includes 3 different color themes (Retro Gold, Neon Berry, Natural Light).
- Undo Functionality: Users can undo the last added transaction.
- Reset Option: Feature to completely reset the wallet and clear database records.
- Instant Calculation: Automatically updates the wallet balance based on income/expense entries.
